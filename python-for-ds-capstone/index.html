Python for Data Science Capstone Project

My Python for Data Science Capstone Project utilizes Python Pandas, NumPy, and Matplotlib to clean and analyze a messy CSV dataset of over 4,500+ entries. This project started from raw data, progressing into a structured and comprehensible dataset using dataframes. Insights were discovered from the dataset that may support important company decisions. This project demonstrates the handling, cleaning, and interpretation of substantial data volumes, allowing me to discover actionable insights.
